Task Management App Setup Guide (React + FastAPI + MongoDB Cloud)
==================================================================

This guide explains how to run the full-stack Task Management app using React, FastAPI, and MongoDB Atlas (cloud database).

# ✅ Task Management App

A full-stack Task Management application built using:

- ⚛️ React (Frontend)
- ⚙️ FastAPI (Backend)
- ☁️ MongoDB Atlas (Cloud Database)

## 🚀 Features

- Add, edit, and delete tasks
- Mark tasks as complete or pending
- Store tasks in the cloud using MongoDB Atlas
- Clean UI and responsive design
- RESTful API backend with FastAPI

Project Structure:
------------------
task-management/
│
├── backend/              # FastAPI backend
│   ├── app/              # FastAPI logic (models, routes, db)

Prerequisites:
--------------
- Node.js and npm
- Python 3.8+
- MongoDB Atlas account


## 🧰 Setup Instructions
------------------------

### 1. 📁 Clone the Repository

```bash
git clone https://github.com/shanvas-ty/task-management.git

cd task-management


### 2.Backend Setup (FastAPI + MongoDB Cloud):
----------------------------------------
1. Navigate to backend:

   cd backend

2. Create virtual environment:

   python -m venv venv

   # For Windows:
   venv\Scripts\activate

   # For macOS/Linux:
   source venv/bin/activate

3. Install Python dependencies:
   pip install -r requirements.txt

4. Add MongoDB Atlas connection string:
   The MongoDB Atlas connection string is already hardcoded in main.py or app/database.py.
   Ensure your database credentials and IP whitelist settings are correct on MongoDB Atlas.

5. Run FastAPI server:
   uvicorn main:app --reload
       or
   uvicorn main:app --reload --port 8000 --host 0.0.0.0

   Visit Swagger docs: http://localhost:8000/docs


### 3.Frontend Setup (React):
-----------------------------
1. Navigate to frontend:
   cd ../frontend            

2. Install npm packages:
   npm install

3. Start React dev server:
   npm start
   App runs at: http://localhost:3000


Connect Frontend with Backend:
------------------------------
Ensure React uses the FastAPI base URL for API calls:
fetch("http://localhost:8000/tasks")

CORS Configuration in FastAPI (main.py):
----------------------------------------
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

You're all set!
