### 2.Backend Setup (FastAPI + MongoDB Cloud):
----------------------------------------
1. Navigate to backend:

   cd backend

2. Create virtual environment:

   python -m venv venv

   # For Windows:
   venv\Scripts\activate

   # For macOS/Linux:
   source venv/bin/activate

3. Install Python dependencies:
   pip install -r requirements.txt

4. Add MongoDB Atlas connection string:
   The MongoDB Atlas connection string is already hardcoded in main.py or app/database.py.
   Ensure your database credentials and IP whitelist settings are correct on MongoDB Atlas.

5. Run FastAPI server:
   uvicorn main:app --reload
       or
   uvicorn main:app --reload --port 8000 --host 0.0.0.0

   Visit Swagger docs: http://localhost:8000/docs


### 3.Frontend Setup (React):
-----------------------------
1. Navigate to frontend:
   cd ../frontend            

2. Install npm packages:
   npm install

3. Start React dev server:
   npm start
   App runs at: http://localhost:3000


Connect Frontend with Backend:
------------------------------
Ensure React uses the FastAPI base URL for API calls:
fetch("http://localhost:8000/tasks")

CORS Configuration in FastAPI (main.py):
----------------------------------------
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
